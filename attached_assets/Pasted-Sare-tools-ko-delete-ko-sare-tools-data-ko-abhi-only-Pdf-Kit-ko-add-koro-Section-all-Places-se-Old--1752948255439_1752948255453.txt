@app.route('/merge', methods=['POST']) def merge_pdf(): files = request.files.getlist('pdfs') merger = PyPDF2.PdfMerger() for file in files: merger.append(file) merger.write("output.pdf") return send_file("output.pdf", as_attachment=True)

🔄 Why NOT One Single /process Route?

❌ One /process	✅ Separate Routes

Confusing logic	Clear, separate files
Long conditions (if tool == 'merge': ...)	Clean functions
Hard to maintain	Easy to debug
Future add-ons = messy	Easy modular integration
